# faceclassify 是一款根据人脸相识度分类照片的软件，其内置高精度人脸识别，检测算法，同时提供了简便的可视化界面，帮助用户快速分类图片


## 1 下载
   [百度云盘地址](https://pan.baidu.com/s/1Qg2r4Emj4uH2Mdr5sOxjfQ)  提取码：27jr

## 2 本地安装
   ### 下载后，本地文件为main.rar,解压main.rar,本地出现main文件夹
   ![download](https://github.com/aifaceclassify/faceclassify/blob/main/images/download.JPG?raw=true)
   ### 进入main文件夹里面，点击 “main.exe”, 稍等一会出现软件主界面
   ![run](https://github.com/aifaceclassify/faceclassify/blob/main/images/run.JPG?raw=true)
   ![main](https://github.com/aifaceclassify/faceclassify/blob/main/images/main.JPG?raw=true)
   
## 3 软件使用教程
   ### （1）配置系统参数。点击设置，弹出“设置”对话框，并 设置“选择人脸库”，“选择输出文件夹”和“输入文件夹”路径
   ![setting](https://github.com/aifaceclassify/faceclassify/blob/main/images/setting.JPG?raw=true)
   ### （2）导入分类图片。点击“导入”按钮,导入需要分类的所有照片, 导出成功后，主界面总数量会更新
   ![setting](https://github.com/aifaceclassify/faceclassify/blob/main/images/settingimport.JPG?raw=true)
   ![setting](https://github.com/aifaceclassify/faceclassify/blob/main/images/settingimportend.JPG?raw=true)
   ### （3）添加分组。点击“添加分组”，填写组名   
   ![addgroup](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAdd.JPG?raw=true)
   ![addgroup1](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAdd1.JPG?raw=true)
   ### （4）分组添加人脸。点击人脸库管理, 弹出 “注册人脸”对话框，点击“添加”，弹出“剪切获取人脸”对话框，接着点击“人脸检测”，等待人脸检测结束后点击人脸
   ![groupAddFace](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAddFace.JPG?raw=true)
   ![groupAddFace1](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAddFace1.JPG?raw=true)
   ![groupAddFace2](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAddFace2.JPG?raw=true)
   ![groupAddFace3](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAddFace3.JPG?raw=true)
   ![groupAddFace4](https://github.com/aifaceclassify/faceclassify/blob/main/images/groupAddFace4.JPG?raw=true)

   ### （5）AI自动分组。进入主界面点击“AI识别”，左下角显示进度，运行结束后，“已归类的tab页”显示人脸相识度大于阀值的人脸（默认值80）
   ![airun](https://github.com/aifaceclassify/faceclassify/blob/main/images/airun.png?raw=true)
   ![airun1](https://github.com/aifaceclassify/faceclassify/blob/main/images/airun1.png?raw=true)

   ### （6）手动调整人脸相似度阀值。可以查看“未归类”人脸相似度值，设置合适的人脸相似度阀值
   ![no_tap](https://github.com/aifaceclassify/faceclassify/blob/main/images/no_tap.png?raw=true)
   ![scoresetting1](https://github.com/aifaceclassify/faceclassify/blob/main/images/scoresetting1.JPG?raw=true)
   ![scoresetting](https://github.com/aifaceclassify/faceclassify/blob/main/images/scoresetting.png?raw=true)

   ### （7）手动调整分组。对于一些复杂的场景，需要手动调整分组
   ![move](https://github.com/aifaceclassify/faceclassify/blob/main/images/move.png?raw=true)

## 4 配置说明
   ### （1）最大线程数：默认为最大cpu物理核数目的2/3,设置越大运行越快，要求硬件性能高
   ### （2）人脸阀值默认0.5，数值越大，人脸检测质量越高
   ### （3）相识度显示阀值，默认为0，控制相识度是否显示框
   ### （4）识别排序，显示度排序
   ### （5）识别模式，对于非正面拍摄的图片，需要旋转一些角度后再进行人脸检测和相似度对比
